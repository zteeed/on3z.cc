## Deployment

```bash
export POSTGRES_PASSWORD=...
docker-compose build --no-cache
docker-compose up -d
```

## Database

```bash
docker exec -it llilgq_db_1 psql -U postgres -d docker -c "select * from short_url_maps;"
```
